# Taschenrechner

Ein einfacher Kommandozeilen-Taschenrechner in Python.

## Funktionen

- Addition, Subtraktion, Multiplikation, Division
- Fehlerbehandlung bei Division durch Null
- Eingabevalidierung

## Starten
```bash
python3 rechner.py
```
